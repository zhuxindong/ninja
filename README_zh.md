# ninja

逆向工程的 `ChatGPT` 代理（绕过 Cloudflare 403 Access Denied）

> 由于`ArkoseLabs`不断地更新修复，项目将应用闭源补丁进行发布，如果你担心安全问题，请你不要使用它。

### 特性

- ChatGPT WebUI
- API密钥获取
- `电子邮件`/`密码`帐户认证
- 代理 `ChatGPT-API`/`OpenAI-API`/`ChatGPT-to-API`
- 支持IP代理池
- 极少的内存占用
